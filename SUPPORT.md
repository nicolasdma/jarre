# Support

Use the right channel so requests are handled quickly.

## Questions and Usage Help

- Open a GitHub Discussion (preferred for how-to and architecture questions).
- If Discussions are not enabled, open an issue using the feature request template.

## Bug Reports

- Use the bug report issue template.
- Include reproduction steps, expected behavior, and logs/screenshots.

## Security Issues

- Follow [SECURITY.md](./SECURITY.md).
- Do not disclose vulnerabilities in public issues.

## Commercial or Partnership Requests

- Open an issue with context and maintainers will route it to the right channel.
